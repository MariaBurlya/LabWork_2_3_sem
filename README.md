# LabWork_2_3_sem

Заданы имена входного текстового файла (существует на диске) и выходного файла (создается программой). Прочитать все строки из входного файла и заполнить контейнер (multimap<int len, int pos, Compare>, где  len – длина строки, pos – позиция начала строки во входном файле (не номер строки!), Compare – соответствующая структура сравнения) информацией. Используя контейнер с информацией о строках записать в выходной файл строки из входного файла в порядке уменьшения длины, причем записаны должны быть только строки с длиной от 20 до 10 символов.
Обратить внимание на функции контейнера multimap  lower_bound и upper_bound.
